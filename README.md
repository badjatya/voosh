# Voosh Assignment

## Installation

1. Install package in BE
   `npm i`

2. Change Directory to FE
   `cd client`

3. Install package in FE
   `npm i`

4. Run FE
   `cd client && npm run dev`

5. Run BE
   `npm run dev`

## Env

-   Create .env file

```
PORT=

DB_URL=

COOKIE_TIME=
JWT_SECRET_KEY=
JWT_EXPIRY=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

-   Create .env.local file in client

```
BE_URL=
NEXT_PUBLIC_GOOGLE_CLIENT_ID=
```
